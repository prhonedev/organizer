Desarrollo Frontend

Frontend es el desarrollo de sitios y aplicaciones web en el navegador, en donde JavaScript juega un papel importante para el dinamismo.

Para sitios web que requieran mayor interactividad, se pueden desarrollar SPAs (Single Page Application), los cuales son sitios web que manejan muchos más estados y datos cambiantes a través del tiempo de forma interactiva por el usuario.

Estos remueven complejidad en el backend y la agregan al frontend mediante JavaScript.

Herramientas Base

Herramientas que complementan el desarrollo de proyectos frontend:

  • Node.js - Plataforma de JavaScript
  • NPM - Administrador de paquetes de Node
  • Git - Administrador de proyectos

Herramientas de Desarrollo

  • Browserify - Administrador de paquetes en el cliente
  • jQuery - Biblioteca de objetivo general
  • Underscore - Funciones utilitarias
  • Backbone - Modelos, vistas y routers para aplicaciones
